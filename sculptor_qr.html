Network Guide
Low latencyLAN / same Wi-Fi
Run a simple local server in the 3d_print_studio folder: python -m http.server 8080 Your LAN IP is detected below. Devices on the same network scan and download instantly.

High latencyInternet / cellular / remote
Host sculptor.html on GitHub Pages, Netlify, or any CDN. Use the Compressed download button — it gzip-compresses the file in your browser before delivery, cutting transfer size by ~70%. Recipients decompress with any archive tool.

Both modes support progress tracking and fall back gracefully if streaming is unavailable.
