How to get a working URL
A Same Wi-Fi (local network)
Open a terminal in your 3d_print_studio folder and run:
python -m http.server 8080 Your LAN IP appears above automatically. Anyone on the same Wi-Fi can scan and open Sculptor instantly.
B Internet (free, permanent hosting)
Go to netlify.com/drop, drag sculptor.html onto the page, and Netlify gives you a permanent public URL in ~30 seconds. Paste that URL above.
C GitHub Pages
Upload sculptor.html to a GitHub repo, enable Pages in Settings → Pages → Deploy from branch. Your URL is https://username.github.io/repo/sculptor.html.
