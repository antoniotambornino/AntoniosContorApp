Installation Instructions
1. Tap QR or download button below → subscribe to unlock
2. thermal_guard.ino → open in Arduino IDE → select your board → Upload
3. printer_monitor.py → run with python printer_monitor.py
4. Connect via USB serial (115200 baud) · Monitor alerts in real time
5. Configure alert thresholds in config.json
